Learn Direx Commands

Direx is built around simple commands to aid your Minecraft command block journey. Here are some core commands to get you started:

  • /syntax <command> - Shows the precise syntax for a Minecraft command.
  • /help <command> - Explains what a specific Minecraft command does and how to use it.
  • /direx help - Provides an overview of all Direx's functionalities.

Privacy Policy for Direx

Last Updated: 8th June, 2025

This Discord bot (Direx) operates with utmost respect for your privacy. Direx does not collect, store, or share any personal user data. Your trust is our priority.

1. How Direx Operates (No Data Collection)

  • The Bot operates entirely within Discord's environment, focusing purely on in-server functionality without external databases.
  • Direx is designed to not collect or log messages, usernames, or any personal identifying information from users.
  • No user-specific data is stored persistently on any disk or database.

2. Data Handling: What Happens When You Use Direx?

  • Any data processed by Direx (such as temporary message content for command parsing, or Discord User/Guild IDs to deliver a command's response) is handled **ephemerally in-memory**.
  • This ephemeral data is used solely for the immediate execution of a command and is **immediately discarded** afterwards.
  • No command inputs, user IDs, or guild IDs are stored beyond the immediate processing required for functionality.

3. Third-Party Data Sharing: None

  • Direx upholds a strict "no sharing" policy.
  • The Bot does not share any user data or interaction data with any third-party services, advertisers, or analytics platforms.
